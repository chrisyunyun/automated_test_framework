# 易威行 OA 系统测试用例文档

- **系统地址**: `https://192.168.16.4:8900`
- **测试框架**: Python + Pytest + Playwright
- **用例来源**: `表格_20260409.csv`

## 一、登录功能测试 (test_oa_login.py)

### LOGIN-FT-001 | 正确账号密码登录成功
- **标记**: smoke
- **前置条件**: 使用正确的 OA 账号和密码
- **测试步骤**: 输入正确账号 → 输入正确密码 → 点击登录
- **预期结果**: 登录成功，跳转至 dashboard 页面，显示用户名

### LOGIN-FT-002 | 手机号为空点击登录
- **标记**: smoke
- **前置条件**: 无
- **测试步骤**: 密码框输入密码 → 点击登录
- **预期结果**: 提示"手机号码"相关错误，停留在登录页

### LOGIN-FT-003 | 密码为空点击登录
- **标记**: smoke
- **前置条件**: 无
- **测试步骤**: 账号框输入账号 → 点击登录
- **预期结果**: 提示"密码"相关错误，停留在登录页

### LOGIN-FT-004 | 账号不存在
- **标记**: regression
- **前置条件**: 使用不存在的账号
- **测试步骤**: 输入不存在的账号 → 输入任意密码 → 点击登录
- **预期结果**: 提示账号不存在，停留在登录页

### LOGIN-FT-005 | 密码错误
- **标记**: regression
- **前置条件**: 使用正确账号但错误密码
- **测试步骤**: 输入正确账号 → 输入错误密码 → 点击登录
- **预期结果**: 提示密码错误，停留在登录页

### LOGIN-FT-006 | 记住登录状态
- **标记**: regression
- **前置条件**: 无
- **测试步骤**: 勾选"记住登录状态" → 输入正确账号密码 → 关闭浏览器 → 重新打开登录页
- **预期结果**: 无需重新登录，自动保持登录状态

### LOGIN-FT-007 | 账号长度过短
- **标记**: regression
- **前置条件**: 无
- **测试步骤**: 输入过短的账号（如 1 位数字） → 输入任意密码 → 点击登录
- **预期结果**: 提示账号格式错误

### LOGIN-FT-008 | 账号非数字
- **标记**: regression
- **前置条件**: 无
- **测试步骤**: 输入非数字账号（如 "abc"） → 输入任意密码 → 点击登录
- **预期结果**: 提示账号格式错误

### LOGIN-FT-010 | Enter 键触发登录
- **标记**: smoke
- **前置条件**: 无
- **测试步骤**: 输入正确账号 → 输入正确密码 → 密码框按 Enter 键
- **预期结果**: 登录成功，跳转至 dashboard 页面

## 新增用例规范

每新增一个测试用例，必须同时更新本文档：

1. 在对应的功能模块章节下添加新条目
2. 填写用例 ID、名称、标记、前置条件、测试步骤、预期结果
3. 更新"用例统计"表格
4. 用例命名规范：`TEST_<模块>_<序号>`

## 运行测试

```bash

pytest

# 运行冒烟测试
pytest -m smoke

# 运行特定文件
pytest tests/test_oa_login.py -v

# 运行特定用例
pytest tests/test_oa_login.py::TestOALogin::test_ft001_login_success -v
```
